# Ari the Fox vs. Ranger Nithy 🦊💩

A tiny browser arcade game. You are Ari the Fox. Your mission: poop all over the park. Ranger Nithy's mission: stop you. You have 90 seconds.

**Play it here:** [https://queenkopaki.github.io/Ari-Fox-Poop-Game/]

## How to play

| Input | Action |
|---|---|
| Arrow keys / WASD | Move Ari |
| Space | Poop |
| Mobile: drag | Move Ari |
| Mobile: tap Ari (or the Poop button) | Poop |

- Every poop is a point. Rack up as many as you can before the timer hits zero.
- Ranger Nithy chases you. If she catches you, you're stunned and she gets faster and angrier.
- Trees and rocks block movement — use them to shake her off.
- Ari refuses to poop in the pond. Ari has standards.
- Your best score is saved in your browser.

## Running it

No install, no build step. It's one HTML file.

## Tech

Vanilla JavaScript and a `<canvas>` — no libraries, no dependencies. All the drawing (Ari, Nithy, trees, poops) is hand-coded canvas shapes in `index.html`.
